🎮 How to Play

- Use **Arrow Left ← / A** to move left  
- Use **Arrow Right → / D** to move right  
- Land on platforms to keep jumping  
- If you fall off the screen, the game is **over**  
- Press **Spacebar** to **restart** after Game Over  

**Tip:** Platforms move down as you go up, so keep climbing!  

---

## 💾 Download and Play

1. Click the green **Code** button at the top of the repo  
2. Select **Download ZIP**  
3. Extract the ZIP file to a folder on your computer  
4. Open index.html in your browser to play
